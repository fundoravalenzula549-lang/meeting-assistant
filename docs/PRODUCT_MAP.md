# 产品地图：个人/小团队在线视频会议转录工作台

## 一句话定位

给个人和小团队使用的会议实时字幕、双轨录音、会后整理工作台。它不依赖会议平台 API，而是通过本机系统音频和麦克风捕获，覆盖腾讯会议、Google Meet、Zoom、飞书、Teams 等在线视频或在线语音场景。

## 用户故事路径

### 1. 会前准备

- 作为会议参与者，我希望检测到可用的系统音频设备和麦克风，避免开会后才发现没有声音。
- 作为会议主持人，我希望选择会议主题、语言和术语提示，让转录结果更贴合上下文。
- 作为隐私敏感用户，我希望默认只监听本机、本地保存数据，并且远程访问必须鉴权。
- 作为跨语种用户，我希望提前选择中文、英文、日文识别，以及英中、日中、中英、中日翻译方向。

### 2. 开始会议

- 作为用户，我希望点一次开始，就同时启动系统音频、麦克风采集、实时转录和录音。
- 作为用户，我希望它支持任意会议平台，因为声音来自操作系统和麦克风，而不是平台插件。
- 作为 Mac 用户，我希望能选择 BlackHole、Loopback 或聚合设备作为系统音频输入。

### 3. 会中实时观看

- 作为参会者，我希望实时看到“我方/对方”双轨字幕，快速区分自己和会议其他人的发言。
- 作为双语会议参与者，我希望看到原文和译文，减少切换上下文的成本。
- 作为演示或访谈用户，我希望有桌面悬浮字幕，不必一直盯着 WebUI。
- 作为操作中用户，我希望能暂停、静音某一轨、恢复，并且断线后 WebSocket 自动重连。

### 4. 会中记录

- 作为用户，我希望会中同步保留系统音频轨、麦克风轨、混合轨，方便复查和二次处理。
- 作为记录者，我希望每个会议自动生成独立 Session 目录，避免不同会议的数据混在一起。
- 作为团队成员，我希望文件名被清洗，避免标题里的特殊字符破坏路径或下载。

### 5. 多人识别

- 作为第一版用户，我需要至少区分“我”和“会议其他人”。
- 作为进阶用户，我希望会议其他人的声音能标成 Speaker 1/2/3，先满足基础复盘，不追求完美身份绑定。
- 作为后续版本用户，我希望能给 Speaker 命名，并在整场会议里保持一致。

### 6. 会后整理

- 作为会议记录者，我希望会议结束后自动校正逐字稿、生成摘要、行动项、决策项和待确认问题。
- 作为知识沉淀用户，我希望导出 TXT 和录音文件，后续可以放进知识库、CRM 或项目管理工具。
- 作为复盘用户，我希望在 WebUI 里打开历史 Session，查看结果和下载文件。

## 第一版功能清单

### 已纳入第一版

- 任意会议平台支持：通过系统音频 + 麦克风双输入实现。
- 系统音频实时转录：会议里其他人的声音。
- 麦克风实时转录：我的声音。
- 双轨字幕：对方 / 我方。
- 会中录音：`system.wav`、`mic.wav`、`mixed.wav`。
- 本地 ASR：faster-whisper。
- 远端 ASR：OpenAI Whisper 风格的 GPU 转录 API。
- 语言：中文、英文、日文、自动检测。
- 翻译：英中、日中、中英、中日。
- 主题和术语增强：作为翻译和会后 AI 的上下文输入。
- 会后 AI：校正逐字稿、摘要、行动项、决策项、待确认问题。
- 导出：`exports/transcript.txt`、录音 WAV、会后 Markdown 纪要。
- WebUI：实时字幕、设备选择、开始/暂停/停止/静音、会后文件打开。
- 桌面悬浮字幕：可选 PyQt6 客户端。
- 设备管理：检测、选择、静音、暂停。
- 断线恢复：WebSocket 客户端重连。
- 基础说话人识别：我方、对方、Speaker 1；预留 Speaker 1/2/3 diarization。
- 安全默认：本机监听、token 鉴权、文件名清洗、Session 隔离。

### 第一版暂不强行承诺

- 不直接接入腾讯会议、Google Meet、Zoom、飞书、Teams 的私有会议 API。
- 不保证第一版完成真实身份级 diarization，只先做来源轨和基础 Speaker 标签。
- 不做云端账号体系、团队空间和权限分层。
- 不做移动端客户端。

## 推荐后续版本

### v1.1 稳定性增强

- 音频设备向导：自动提示 BlackHole/Loopback/聚合设备配置。
- 转录失败重试队列和更细的错误提示。
- Session 搜索、重命名、删除。
- 更完整的字幕导出：SRT、VTT、Markdown。

### v1.2 多人识别增强

- 对方轨道离线 diarization。
- Speaker 命名和手动合并。
- 跨 Session 的常见说话人声纹记忆。

### v1.3 团队协作

- 局域网安全分享。
- 项目/客户维度归档。
- Notion、飞书文档、Google Docs、Slack/Teams 导出。
